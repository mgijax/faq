How do I search or browse PIRSF protein superfamilies mapped to mouse, human, and rat genes?
Overview
You can use the Protein Superfamily Vocabulary Browser to retrieve genes, sequences and human, rat and mouse orthologs for a given superfamily.
Accessing the Protein Superfamily Vocabulary Browser
On most MGI pages, you can access the Protein Superfamily Vocabulary Browser from the Search menu as shown in the image at right.

For this tutorial, open the WIFetch?page=pirsfVocab&subset=A' onClick="javascript:openNewWindow('WIFetch?page=pirsfVocab&subset=A'); return false;"> Protein Superfamily Vocabulary Browser in a new window.


Example 1. Finding zinc finger superfamily members
  1. Type zinc finger in the text box.

  2. Click Search.

Example 2. Finding protein superfamily names starting with the letter M
  1. Click the link to the letter WIFetch?page=pirsfVocab&subset=M' onClick="javascript:openNewWindow('WIFetch?page=pirsfVocab&subset=M'); return false;"> M to retrieve all protein superfamily names starting with this character.
  1. Click any link to a protein superfamily name to retrieve a list of genes within this superfamily.
