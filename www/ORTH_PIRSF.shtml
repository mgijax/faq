How do I search or browse PIRSF protein superfamilies mapped to mouse, human, and rat genes?

  1. Click WIFetch?page=pirsfVocab&subset=A"TARGET = "BLANK">here to open the Protein Information Resource SuperFamily (PIRSF) Browser in a new window. The PIRSF Browser is located beneath the Search menu, as shown below.

  2. On the PIRSF Protein Superfamilies browser page, type zinc finger into the text box.

  3. Click the Search button.

  4. The search results display a list of protein superfamilies containing the phrase "zinc finger". Click the link to a protein superfamily name to retrieve a list of genes within this superfamily.

  5. To browse protein superfamily names, click the link to the first character of a protein superfamily name at the top of the page. For example, click the link to the letter B to retrieve all protein superfamily names starting with this character.

  6. Click any link to a protein superfamily name to retrieve a list of genes within this superfamily.
