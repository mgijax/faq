How do I find strains with RFLP/PCR polymorphisms in a particular gene?

  1. Click here to open the RFLP/PCR Polymorphism Query Form in a new window. The RFLP/PCR Polymorphism Query Form is located beneath the search menu as shown below.

  2. You may use MGI to compare polymorphisms between mouse strains. MGI contains single nucleotide polymorphisms (SNPs), restriction fragment length polymorphisms (RFLPs), and PCR polymorphisms for a long list of strains.

  3. Type Cyp1a1 into the Locus Symbol/Name text box.

  4. Click the Search button.

  5. Click a PCR link in the Type column to view details about alleles, fragment sizes and strains.

  6. Click a link in the Probe column to retrieve further detail, including primer sequences in many cases.

  7. For RFLPs, return to the search results for Cyp1a1 and click an RFLP link in the Type column to view details about alleles, fragment sizes and strains.
