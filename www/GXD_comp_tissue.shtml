How do I find genes expressed in one tissue but not another?
Overview
You can use the Gene Expression Data Expanded Query Form to query for genes expressed in some anatomical structures and/or developmental stages but not in others.
Accessing the Gene Expression Data Expanded Query Form
On most MGI pages, you can access the Gene Expression Data Expanded Query Form from the Search menu as shown in the image at right.

For this tutorial, open the WIFetch?page=expressionQFexpanded' onClick="javascript:openNewWindow('WIFetch?page=expressionQFexpanded'); return false;"> Gene Expression Data Expanded Query Form in a new window.


Example 1. Finding genes expressed in the forebrain but not in the midbrain
  1. Type forebrain into the Anatomical Structures(s) text box in the upper left quadrant of the Expression section of the form. This part of the search will return assays in which expression was detected.

  2. Type midbrain into the Anatomical Structures(s) text box in the lower left quadrant of the Expression section. This part of the search will return assays in which expression was NOT detected.

  3. Click Search.
  1. Click a link in the Result Details column to view assay details. Each gene may be listed more than once, since the structures you have searched for may appear in more than one assay.
