How do I view annotations for mouse/human/rat ortholog sets?
Overview
You can use the Search Box to find information about genes, markers or accession identifiers in MGI. The Search Box returns a list of genes, markers, or accession IDs that contain your entry.
Accessing the Search Box
On most MGI pages, you can find the Search Box in the upper right corner, as shown in the image.

For this tutorial, open the homepages/index.shtml' onClick="javascript:openNewWindow('homepages/index.shtml'); return false;"> MGI home page in a new window.


Example 1. Finding orthologs for gene Pax6
  1. Type pax6 into the Search Box.

  2. Click Quick Search.
  1. The Search Results page lists symbols and names containing Pax6. Click the link to WIFetch?page=markerDetail&key=12184' onClick="javascript:openNewWindow('WIFetch?page=markerDetail&key=12184'); return false;"> Pax6.
  1. Click the link to searches/homology_report.cgi?_Marker_key=12184' onClick="javascript:openNewWindow('searches/homology_report.cgi?_Marker_key=12184'); return false;"> Mammalian Orthology on the Pax6 Gene Detail page.
  1. The Mammalian Orthology page supplies a table of Pax6 orthologs in human, mouse and rat. To view a graph of Gene Ontology annotations for these orthologs, click the link to WIFetch?page=GOOrthologyGraph&id=MGI:97490' onClick="javascript:openNewWindow('WIFetch?page=GOOrthologyGraph&id=MGI:97490'); return false;"> Mouse-Human-Rat Comparative GO Graph.
