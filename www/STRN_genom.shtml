How do I find genomic sequence for a specific mouse strain?

  1. Click here to open the Mouse Sequence Query Form in a new window. The Mouse Sequence Query Form is located beneath the search menu as shown below.

  2. For Sequence Type, select DNA from the pick list.

  3. Find the Sequence source section. Type 129 into the Strain/Species text box.

  4. Find the Map position section and select chromosome X from the pick list.

  5. Click the Search button.

  6. The search will return all DNA sequences on chromosome X that originate from strain 129 or substrains. Check a box in the Select column.

  7. Find the drop-down menu in the upper center of the page that allows you to download in FASTA format or forward to MouseBLAST.

  8. Click the Go button to download the selected sequence in FASTA format.
