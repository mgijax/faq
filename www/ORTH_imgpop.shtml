Mammalian Orthology in MGI

MGI provides a curated set of mammalian orthologs from sequenced mammalian genomes (e.g., human, rat, dog, chimp). Major ways to access orthology data include:

  • Using the link on gene detail pages
    A summary of organisms with orthologs for a mouse gene can be accessed by clicking on the Mammalian Orthology link in the Mammalian homology section of that gene's detail page. For detailed instructions see the following:
