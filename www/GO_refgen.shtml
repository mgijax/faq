How do I download the list of all GO Consortium Reference Genome Project mouse genes?
Overview
You can download plain text files from the MGI Data and Statistical Reports page. These files include lists of all genes and markers in MGI, alleles of genes and markers, annotations to genes and alleles and more.
Accessing the MGI Data and Statistical Reports Page
On most MGI pages, you can access the MGI Data and Statistical Reports Page from the Download menu as shown in the image at right.

For this tutorial, open the MGI Data and Statistical Reports Page in a new window. Scroll down this page for further instructions.


Example 1. Downloading all GO Consortium Reference Genome Project mouse genes
  1. Click the link to Gene Ontology Data to move down the page.
  1. Click the link to go_ref_genome.mgi to download a plain text file containing all GO Consortium Reference Genome Project mouse genes in MGI.
