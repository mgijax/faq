How do I find expression data for a specific gene?

  1. Click homepages/index.shtml"TARGET = "BLANK">here to open the MGI home page in a new window.

  2. Type a gene symbol into the search box in the upper right hand corner of the web page. As an example, use Pax6.

  3. Click the Search button.

  4. On the Search Results page, click the link to Pax6 in the Symbol, Name column.

  5. Find the Expression section on the Pax6 Gene Detail page. This section summarizes Pax6 expression data in MGI:

    • by Theiler (developmental) Stage;
    • by Tissue;
    • by Assay Type.

  6. For each Assay Type, the number of results and number of assays is supplied. To view all results of Western blots, for example, click on the numerical link beneath Results for Assay Type Western blot. Then click a link in the Results Details column to view assay details.

  7. The Expression section also lists images, references, and cDNA clones associated with Pax6. Click the numerical links next to Images, GXD Literature Content Summary, and cDNA source data to view these.
