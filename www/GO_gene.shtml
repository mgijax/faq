How do I find all GO annotations for a specific gene?


Overview
You can use the Search Box to find information about genes, markers or accession identifiers in MGI. The Search Box returns a list of genes, markers, or accession IDs that contain your entry.
Accessing the Search Box
On most MGI pages, you can find the Search Box in the upper right corner, as shown in the image.

For this tutorial, open the homepages/index.shtml' onClick="javascript:openNewWindow('homepages/index.shtml'); return false;"> MGI home page in a new window.


Example 1. Finding orthologs for gene Pax6
  1. Type pax6 into the search box.

  2. Click Search.
  1. The Search Results page lists symbols and names containing Pax6. Click the link to WIFetch?page=markerDetail&key=12184'target="blank"> Pax6.
  1. Click the link to WIFetch?page=markerGO&key=12184'target="blank"> All GO classifications(44) on the Pax6 Gene Detail page.
  1. The Gene Ontology classifications page supplies all classifications in text, table and graphical form. Click the link to WIFetch?page=markerGO&key=12184#tabular'target="blank"> Tabular View to view a table listing all classifications, or the link to WIFetch?page=GOMarkerGraph&id=MGI:97490'target="blank"> GO Graph to view a graph of all classifications.
