How do I find all GO annotations for a specific gene?

  1. Click homepages/index.shtml"TARGET = "BLANK">here to open the MGI home page in a new window.

  2. Type a gene symbol into the search box in the upper right hand corner of the web page. As an example, use Pax6.

  3. Click the Search button.

  4. On the Search Results page, click the link to Pax6 in the Symbol, Name column.

  5. Find the Gene Ontology (GO) classifications section on the Pax6 Gene Detail page.

  6. Click the link to All GO classifications(x).

  7. The Gene Ontology classifications page supplies all classifications in text, table and graphical form. Click the link to Tabular View to view a table listing all classifications, or the link to GO Graph to view a graph of all classifications.
