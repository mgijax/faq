How do I find expression data for a specific gene and tissue?
Overview
You can use the Gene Expression Data Query Form to search for detailed expression data of greatest interest to you using a variety of query parameters.
Accessing the Gene Expression Data Query Form
On most MGI pages, you can access the Gene Expression Data Query Form from the Search menu as shown in the image at right.

For this tutorial, open the WIFetch?page=expressionQF' onClick="javascript:openNewWindow('WIFetch?page=expressionQF'); return false;"> Gene Expression Data Query Form in a new window.


Example 1. Finding expression assays in brain for gene Ptch1
  1. Type ptch1 into the Gene/Marker Symbol/Name text box.

  2. Type brain into the Anatomical Structure(s) text box.

  3. Change the Return data type button to Assay Results.

  4. Click Search.
  1. Click on a link in the Result Details column to view details from assays that examined Ptch1 expression in the brain.
