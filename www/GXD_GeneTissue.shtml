How do I find expression data for a specific gene and tissue?

  1. Click WIFetch?page=expressionQF"TARGET = "BLANK">here to go to the Gene Expression Data Query Form. The Gene Expression Data Query Form is located beneath the Search menu in the toolbar, as shown below.

  2. In the Gene section near the top of the form, type the gene symbol of interest into the Gene/Marker Symbol/Name field. As an example, use Ptch1.

  3. Find the Expression section of the form and type the name of a tissue into the Anatomical Structure(s) field. As an example, type brain.

  4. Find the Sorting and output format section at the bottom of the form. Change the Return data type button to Assay Results.

  5. Click the Search button.

  6. Once the query results have been returned, click on any link in the Result Details column to view details from assays that examined Ptch1 expression in the brain.
