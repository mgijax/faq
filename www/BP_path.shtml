How do I compare mouse pathways to human pathways?
Overview
You can use the Pathway Tools Query Page to access mouse genetic and genomic data in the context of physiological and cellular processes.
Accessing the Pathway Tools Query Page
On most MGI pages, you can access the Pathway Tools Query Page from the Search menu as shown in the image at right.

For this tutorial, open the Pathway Tools Query Page in a new window.


Example 1. Comparing human and mouse pathways
  1. Click the link to the Comparative Analysis.
  1. Check the box next to Pathways.

  2. Check boxes next to both Homo sapiens and Mus musculus C57BL/6J.

  3. Click Submit.
