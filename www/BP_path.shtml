How do I compare mouse pathways to human pathways?

  1. Click homepages/index.shtml"TARGET = "BLANK">here to open the MGI home page in a new window.

  2. Find the box labeled MGI News in the lower half of the page.

  3. Click the link to MouseCyc within this box.

  4. On the Pathway Tools Query page, find the link to Comparative Analysis near the bottom of the page. Click this link.

  5. Select which set of comparative analysis tables you wish to generate by checking the box next to Pathways.

  6. Check boxes next to both Homo sapiens and Mus musculus C57BL/6J near the bottom of the page.

  7. Click the Submit button.
