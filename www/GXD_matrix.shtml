How do I interpret the results of my expression search?
Overview

You can use the gxd' onClick="javascript:openNewWindow('gxd'); return false;"> Gene Expression Standard or Differential Data Query in order to find expression data for a tissue, gene or set of genes.

See FAQ below for details on how to:

  • find expression data for a specific gene and tissue? FAQ
  • find expression data for tissue during a specific stage of development? FAQ
  • find genes expressed in one tissue but not another (or not reported in another)? FAQ
  • find references that report expression data for a gene at a specific age? FAQ

Results from your search will be presented in a table with six tabs, each highlighting a different facet or way to assess the data. Filters appear above the table allowing you to refine the data; if applied, these will filter all tabs. The original search terms and parameters are displayed under the Results header as a reminder. A Click to Modify Search button also appears at the top of the page which will expand your original query form on click, allowing additional changes.

For the examples below, a search for all expression data annotated to genes with an association to autism (via OMIM) was run.

The Genes Tab

The Genes tab provides a list of the unique genes with expression data represented in these results. A number in the tab header provides a count of all genes. Only genes with at least one gene expression annotation are reported.

This list can be downloaded (see buttons just above the table) or forwarded to MGI Batch Query where additional data can be added.

To filter your results by gene, use the gene column filters on the Tissue x Gene Matrix tab.

Clicking on Gene Symbols will go to the Gene Detail page within MGI.

The Assays Tab

The Assays tab lists every analysis type and data source used by GXD to make a gene expression annotation. The majority of these are peer-reviewed publications, data from direct submission and batch loads from collaborating large scale projects.

Assay records can be sorted by Assay Type, Gene Symbol, or Reference (by J:#) by using the arrows in column headers, or filtered using the Filter expression by: buttons above the tabs.

Clicking on "data" within the Assay Results tab will go to the total curated data that matches that Assay Type, Gene and Reference. The linked Assay Detail page will contain specific about the specimen(s) used (mouse strain, sex and age where known), probe ID and preparation (where applicable), a text description of the results curated from author statements (qualitative expression patterns and levels) and supporting images, marked by a camera icon.

The Assay Results Tab

The Assays Results tab is the most comprehensive list of the data within GXD. These results can be sorted using the arrows in most column headers, filtered using the "Filter Expression by:" buttons above the tabs, and downloaded using the buttons on the table just above the data.

To filter more precisely by anatomical substructures or by gene, use the row and column filters available on the Tissue x Stage Matrix or Tissue x Gene Matrix respectively.

Each piece of data is accompanied by a reference and additional details can be found by clicking the "data" hyperlink in the Result details column.

Note that a result will be returned for both positive and examined-but-absent gene expression data (see Yes/No in "Detected" column) by default. This can be changed by selecting the radio button corresponding to detected, not detected, or both in the original search parameters, or a filter can be used from above the results table.

The Images Tab

GXD has permission from several journals and resources to reproduce images that support gene expression data annotations. These figures are presented on the Images tab as well as on individual Assay Detail pages.

Metadata accompanies each image with the gene(s) assayed, the assay type and a Result details link that brings you to the Assay Detail page with detailed specimen and protocol information.

Note that some images have expression data for multiple genes.

The Tissue x Stage Matrix Tab

The Tissue x Stage Matrix is an interactive graphical display of gene expression data within GXD, organized into anatomical structures (rows) by Theiler developmental stages (TS; columns). Mouse age in embryonic (E) or post-natal (P) days is indicated above the TS categories.

White boxes in this grid indicate that the tissue x stage is not a valid combination, for example, embryonic structural precursors in adult mice, or organs in early embryos before development and differentiation has occurred. If a circle appears within the box, the tissue x stage is a valid combination, but no expression data has been reported. Follow the link to browse the complete vocab/gxd/anatomy/EMAPA:16039' onClick="javascript:openNewWindow('vocab/gxd/anatomy/EMAPA:16039'); return false;"> Mouse Developmental Anatomy Browser.

Filled boxes indicate that expression data exists: blue if genes are annotated as expressed within structures or substructures, red where genes are annotated as "not detected" within structures or substructures. Heat map shading represents the number of positive or negative expression results underlying the intersection, not level of expression for a gene within a tissue.

Cut corners are used to indicate that negative (red) or ambiguous (gold) results have been reported in substructures. Click on the Legend to see examples.

Use scroll bars to explore down the list of structures. When a triangle appears in front of the term, click to expand substructures. To restrict results by specified stages, structures or substructures, click the box to place a checkmark in selected rows or columns and click the Filter button just above the table on this tab. This will retain selected rows only and filter all tabs.

To see the data at specific Tissue x Stage intersections, click the corresponding filled cell to generate a pop-up (shown for "brain" at "TS20"). A table with the number of genes detected and number of genes not detected, along with the number of results supporting those conclusions will appear. To see the data or images associated with this subset. Click the buttons on the pop-up table, which will act as a filter.



The Tissue x Stage Matrix Tab

The Tissue x Gene Matrix is an interactive graphical display of gene expression data within GXD, organized into anatomical structures (rows) by gene (columns).

White cells in this grid indicate that the tissue x gene combination has no data reported, and filled cells indicate that gene expression has been detected (blue) or examined, but not detected (red) within a given structure or substructure. Heat map shading represents the number of positive or negative expression results underlying the intersection, not level of expression for a gene within a tissue.

Use scroll bars to explore down the list of structures. When a triangle appears in front of the term, click to expand substructures. To restrict results by specified genes, structures or substructures, click the box to place a checkmark in selected rows or columns and click the Filter button just above the table on this tab. This will retain selected rows only and filter all tabs.

To see all anatomical structures and data organization, use the vocab/gxd/anatomy/EMAPA:16039' onClick="javascript:openNewWindow('vocab/gxd/anatomy/EMAPA:16039'); return false;"> Mouse Developmental Anatomy Browser.

To see the data at specific Tissue x Gene intersections, click the corresponding filled cell to generate a pop-up (shown for "Tbx1" in "heart"). A table with the number of results reporting gene expression detected and/or not detected will appear. To see the data or images supporting these with this subset. Click the buttons on the pop-up table, which will act as a filter.
