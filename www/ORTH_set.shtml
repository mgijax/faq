How do I view the ortholog set for a mouse gene?

  1. Click homepages/index.shtml"TARGET = "BLANK">here to open the MGI home page in a new window.

  2. Type a gene symbol into the search box in the upper right hand corner of the web page. As an example, use Pax6.

  3. Click the Search button.

  4. On the Search Results page, click the link to Pax6 in the Symbol, Name column.

  5. Find the Mammalian homology section on the Pax6 Gene Detail page.

  6. Click the link to Mammalian orthology to view a table of Pax6 orthologs in different species. The table includes genomic location, accession identifiers, and evidence codes used to establish Pax6 orthology in the species listed.
