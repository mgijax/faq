How can I contribute my data to MGI?

  1. For experimental data such as mapping, homology, phenotype, expression or other data, select Contribute Data to MGI from the Submit menu, as shown below.

  2. To request nomenclature or lab codes, select Nomenclature & Lab Code Requests from the Submit menu, as shown below.

  3. To review mouse nomenclature, select Nomenclature from the More Resources menu, as shown below.

  4. The Quick Guides are good places to start for gene, allele and mutation nomenclature. More detailed information may be found in the full guides. As an example, allele symbol Egfrtm1Cwr refers to the first targeted mutation (tm1) of the Egfr gene produced by Case Western Reserve University (Cwr).
