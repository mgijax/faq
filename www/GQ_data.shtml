How can I contribute my data to MGI?
Overview
You can submit your experimental data to MGI to make it publicly available for database searches, thereby increasing its value, exposure and usefulness.
Submitting data to MGI
On most MGI pages, click Submit Data in the navigation bar as shown in the image at right.

For this tutorial, open the submit.shtml' onClick="javascript:openNewWindow('submit.shtml'); return false;"> Submit Data page in a new window. Scroll down this page for further instructions.


Submitting allele, phenotype or expression data
  1. For new alleles, mutations, or transgenes, click the link to nomen/allmut_form.shtml' onClick="javascript:openNewWindow('nomen/allmut_form.shtml'); return false;"> Allele Nomenclature to retrieve the Mutation Submission Form. Use this form to describe spontaneous, induced, or genetically engineered mutations. MGI staff will help you assign correct nomenclature and provide an MGI accession identifier.

  2. To submit phenotype information for mutations already in MGI, click the link to nomen/phenotype_data_form.shtml' onClick="javascript:openNewWindow('nomen/phenotype_data_form.shtml'); return false;"> Phenotype Data to retrieve the Phenotype Data Submission Form.

  3. To submit expression data, click the link to GXD/GEN/gxd_submission_guidelines.shtml' onClick="javascript:openNewWindow('GXD/GEN/gxd_submission_guidelines.shtml'); return false;"> Expression Data to retrieve guidelines for Submitting expression data to the Gene Expression Database (GXD).
