How can I contribute my data to MGI?
Overview
You can submit your experimental data to MGI to make it publicly available for database searches, thereby increasing its value, exposure and usefulness.
Submitting data to MGI
On most MGI pages, click Submit Data in the navigation bar as shown in the image at right.

For this tutorial, open the submit.shtml' onClick="javascript:openNewWindow('submit.shtml'); return false;"> Submit Data page in a new window. Scroll down this page for further instructions.


Submitting allele, phenotype or expression data
  1. To describe spontaneous, induced, or genetically engineered mutations, to register new mouse strains, or to describe phenotypes, click the link to submissions/amsp_submission.cgi' onClick="javascript:openNewWindow('submissions/amsp_submission.cgi'); return false;"> Allele, Strain & Phenotype Data.

  2. To submit expression data, click the link to GXD/GEN/gxd_submission_guidelines.shtml' onClick="javascript:openNewWindow('GXD/GEN/gxd_submission_guidelines.shtml'); return false;"> Expression Data to retrieve guidelines for Submitting expression data to the Gene Expression Database (GXD).
