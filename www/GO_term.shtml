How do I find all mouse genes with shared annotation to a GO term?
Overview
You can use the GO Browser to search for genes annotated to a specific Gene Ontology term.
Accessing the GO Browser
On most MGI pages, you can access the GO Browser from the Search menu as shown in the image at right.

For this tutorial, open the searches/GO_form.shtml' onClick="javascript:openNewWindow('searches/GO_form.shtml'); return false;"> GO Browser in a new window. Scroll down this page for further instructions.


Example 1. Finding genes annotated to GO term "neuron migration"
  1. Type the term neuron migration in the text box.

  2. Click Search.
  1. Click the link to the term searches/GO.cgi?id=GO:0001764' onClick="javascript:openNewWindow('searches/GO.cgi?id=GO:0001764'); return false;"> neuron migration.
  1. Click the link to searches/GOannot_report.cgi?id=GO:0001764' onClick="javascript:openNewWindow('searches/GOannot_report.cgi?id=GO:0001764'); return false;"> 56 genes, 62 annotations next to the term neuron migration [GO:0001764].
  1. The results list genes annotated with this term. Notice that the Evidence column includes the codes IDA (inferred from direct assay), IGI (inferred from genetic interaction), and IMP (inferred from mutant phenotype). The final column, labeled Ref(s), provides links to the references used to make these annotations.

  2. Scroll down the page and click the link to WIFetch?page=markerDetail&key=12184' onClick="javascript:openNewWindow('WIFetch?page=markerDetail&key=12184'); return false;"> Pax6.
  1. Click the link to All GO classifications: go/marker/MGI:97490' onClick="javascript:openNewWindow('go/marker/MGI:97490'); return false;"> (112 annotations) on the Pax6 Gene Detail page.
  1. The Gene Ontology classifications page supplies all classifications in text, table and graphical form. Click the link to go/marker/MGI:97490#tabular' onClick="javascript:openNewWindow('go/marker/MGI:97490#tabular'); return false;"> Tabular View to view a table listing all classifications, or the link to WIFetch?page=GOMarkerGraph&id=MGI:97490' onClick="javascript:openNewWindow('WIFetch?page=GOMarkerGraph&id=MGI:97490'); return false;"> GO Graph to view a graph of all classifications.
