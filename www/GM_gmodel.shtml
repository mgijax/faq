How do I see different gene models along with the MGI-curated representative model?
Overview
You can use the Search Box to find information about genes, markers or accession identifiers in MGI. The Search Box returns a list of genes, markers, or accession IDs that contain your entry.
Accessing the Search Box
On most MGI pages, you can find the Search Box in the upper right corner, as shown in the image.

For this tutorial, open the homepages/index.shtml' onClick="javascript:openNewWindow('homepages/index.shtml'); return false;"> MGI home page in a new window. Scroll down this page for further instructions.


Example 1. Displaying Ensembl, NCBI, VEGA, and MGI representative transcripts for gene Pax6
  1. Type pax6 into the Search Box.

  2. Click Quick Search.
  1. The Search Results page lists symbols and names containing Pax6. Click the link to WIFetch?page=markerDetail&key=12184' onClick="javascript:openNewWindow('WIFetch?page=markerDetail&key=12184'); return false;"> Pax6.
  1. Click the MGI Mouse Genome Browser link or thumbnail image on the Pax6 Gene Detail page.
  1. Scroll down the MGI Mouse Genome Browser page to the Tracks section.

  2. Check the boxes next to the following:
    Genes
    Ensembl transcripts
    MGI Representative transcripts
    NCBI transcripts
    VEGA transcripts

  3. Click Update Image.
  1. The Details section displays gene models from sources selected above in step 6.

  2. Click on a gene model to view more information.
