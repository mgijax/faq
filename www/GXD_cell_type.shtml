How do I find expression data for a cell type?
Overview

The Cell Ontology (CL) is a hierarchically structured, controlled vocabulary that describes a broad range of canonical biological cell types. You can use the Cell Ontology (CL) Browser to search for and view descriptions of cell types; to explore the ontology hierarchy and locate specific terms in context. You can then use the links provided to find the expression results associated with those terms.

Accessing the Cell Ontology Browser

On most MGI pages, you can access the Cell Ontology Browser from the Search menu as shown in the image at right.

Alternatively, you can also navigate to the Gene Expression Database (GXD) home page using the tile found on the MGI home page (arrow in image at right). Once there, click on the Cell Ontology Browser tile.

Example. Find all gene expression results annotated to muscle cell and child terms.
  1. Begin typing muscle cell into the Cell Type Search text box and select from the drop down menu options that appear.

    The Cell Type field has an autocomplete list that activates on the second character.

    You may also use a CL numerical identifier in this field.

  1. Select a term from the autocomplete list to activate a search. The search finds all cell type terms that contain any of your search terms and displays a list of those terms.
  1. Click on a term in the search results (muscle cell; arrow) to update the Term Detail and Tree View sections of the browser.

  2. To review expression annotations that use the term or one of its subterms, click on the hyperlinked number of expression results next to the term in the tree view (red box).

    The Term Detail section (upper portion of the right pane) displays information specific for that term, including synonyms and definitions.

    The Tree View (lower portion of the right pane) contains a hierarchical tree view for CL with the selected term highlighted. Finer detail cell type structures are represented as subterms. Scroll and/or click on triangle toggles to navigate around the tree. Greyed out terms have no associated expression annotations; some of these cell types are not present in the mouse.

  1. The expression results appear in a table. The default display tab is Assay results (boxed). The other tabs provide different views of the same data.

  2. The return can be further refined using the filters found at the top of the page; they use attributes of the assay results or curated annotations to the genes analyzed. Three filters have been applied in the example (brackets).

  3. To view the detailed data, click on data (arrow) in the Result Details column.

    As illustrated in the Cell Type column (box), a search using muscle cell returns all annotations using it and its subterms.

    Data from the Assay results and Genes tabs can be exported as plain text by using buttons in the header (arrowhead). Genes can be exported to the MGI Batch Query or MouseMine.

  1. In addition to reporting the results obtained for each specimen, including the level and pattern of expression for each anatomical structure and cell type examined, the Assay Detail page provides information about the gene analyzed, assay type, probes and experimental conditions used, and age and genetic background of the specimen; links to additional details are supplied. Images of the data are displayed when available.
