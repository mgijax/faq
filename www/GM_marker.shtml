How do I search for genes, QTL, or other marker types in a specific region of the mouse genome?

  1. Click here to go to the Genes and Markers Query Form. The Genes and Markers Query Form is located beneath the Search menu, as shown below.

  2. In the Gene/Marker section of the page, find the Type drop-down menu and select Gene.

  3. Find the Map Position section of the form and select 7 from the Chromosome(s) list.

  4. In the text box beneath Genome Coordinates, type 130 and 140.

  5. Change the pull-down menu near the Genome Coordinates text box from bp to Mbp.

  6. Click the Search button. The search results will list all genes on chromosome 7 between 130 and 140 Mbp.

  7. Return to the Genes and Markers Query Form. Change the the Type drop-down menu from Gene to QTL. Repeat the search to retrieve all QTL on chromosome 7 between 130 and 140 Mbp.
