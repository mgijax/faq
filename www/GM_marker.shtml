How do I search for genes, QTL, or other features in a specific genome interval?

Genes and Markers Query Form
You can use the Genes and Markers Query Form to narrow your search to a specific region of the mouse genome by choosing a chromosome number and entering genome coordinates, a cytogenetic band, centiMorgan position, or marker range. Within this interval, you can choose to retrieve genes, QTL, pseudogenes, microRNA, or other marker types.

To begin, click here to open the Genes and Markers Query Form in a new window. The Genes and Markers Query Form is located beneath the Search menu as shown.


Finding genes within a specific genomic interval
  1. In the Gene/Marker section of the page, find the Type drop-down menu and select Gene.

  2. Find the Map Position section of the form and select 7 from the Chromosome(s) list.

  3. In the text box beneath Genome Coordinates, type 130 and 140.

  4. Change the pull-down menu near the Genome Coordinates text box from bp to Mbp.

  5. Click the Search button. The search results will list all genes on Chromosome 7 between 130 and 140 Mbp.

Finding QTL within a specific genomic interval
  1. Return to the Genes and Markers Query Form.

  2. Change the Type drop-down menu from Gene to QTL.

  3. Click the Search button to retrieve all QTL on Chromosome 7 between 130 and 140 Mbp.
