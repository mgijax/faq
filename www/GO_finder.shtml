How do I find shared function for a list of specific mouse genes?
Overview
You can use the GO Functional Annotations page to retrieve functional annotations to mouse genes.
Accessing the GO Functional Annotations page
On most MGI pages, you can access the GO Functional Annotations page by clicking the appropriate tab as shown in the image at right.

For this tutorial, open the function.shtml' onClick="javascript:openNewWindow('function.shtml'); return false;"> GO Functional Annotations page in a new window.


Example 1. Finding shared function for Pax genes
  1. Click the link to the MGI GO Term Finder in the box labeled Access Data.
  1. Type these three gene symbols into the text box:

    Pax6
    Pax3
    Pax2

  2. Select the radio button next to Function.

  3. Select Exclude IEAs.

  4. Click Search.
