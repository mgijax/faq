How do I find shared function for a list of specific genes?

  1. Click here to go to the GO Functional Annotations page. This page may be accessed by clicking the GO Functional Annotations link on the MGI home page, or by clicking the tab near the top center of other MGI pages.

  2. Find the box labeled Access MGI-GO data located at left. In the lower right of this box, click the link to the MGI GO Term Finder.

  3. Using the MGI GO Term Finder, you can enter a list or upload a file of official MGI gene/marker symbols. You can choose to return GO annotations from either the Biological Process, Molecular Function, or Cellular Component Ontology for this set of genes.

  4. As a quick example, type these three gene symbols into the text box for Step 1:

    Pax6
    Pax3
    Pax2

  5. For Step 2, select the radio button next to Function.

  6. For Step 3, select Exclude IEAs.

  7. Click the Search button.
