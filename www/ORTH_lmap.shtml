How do I view a mouse linkage map with human orthologs?
Overview
You can use the Linkage Map Query Form to retrieve a linkage map of a mouse chromosome. You can also view all human/mouse orthologies for a given chromosome.
Accessing the Linkage Map Query Form
On most MGI pages, you can access the Linkage Map Query Form from the Search menu as shown in the image at right.

For this tutorial, open the searches/linkmap_form.shtml' onClick="javascript:openNewWindow('searches/linkmap_form.shtml'); return false;"> Linkage Map Query Form in a new window. Scroll down this page for further instructions.


Example 1. Viewing a linkage map of mouse chromosome 16 with human orthologs
  1. Select 16 from the Chromosome drop-down menu.

  2. Type Serpind1 into the first text box, and Dgcr2 into the second text box to view a linkage map of mouse chromosome 16 between these two genes.

  3. Select human (Homo sapiens) from the Comparative Maps pick list.

  4. Click Search.
  1. The linkage map shows mouse chromosome 16 from 9.50 to 10.37 centiMorgans (cM) between genes Serpind1 and Dgcr2. Click a mouse gene symbol for detailed information about a mouse gene. Human orthologs and cytobands are listed at right in black text.
