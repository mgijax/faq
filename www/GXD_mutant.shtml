How do I find expression data in mouse mutants?
Overview
You can use the Gene Expression Data Search Form to find detailed expression data of greatest interest to you using a variety of query parameters.
Accessing the Gene Expression Data Search Form
On most MGI pages, you can access the Gene Expression Data Search Form from the Search menu as shown in the image at right.

For this tutorial, open the gxd' onClick="javascript:openNewWindow('gxd'); return false;"> Gene Expression Data Search Form in a new window. Scroll down this page for further instructions.


Example 1. Finding expression assays for Shh mutants
  1. Type shh into the Mutant / wild type text box.
  2. Click Search.
  1. Click on any data link in the Result Details column to view details for a specific assay.
The results summary will list assays using Shh mutants. The specific allele(s) used to obtain each result will be listed in the Mutant Allele(s) column.
