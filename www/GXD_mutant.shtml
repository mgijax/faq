How do I find expression data in mouse mutants?

  1. Click WIFetch?page=expressionQF"TARGET = "BLANK">here to open the Gene Expression Data Query Form in a new window. The Gene Expression Data Query Form is located beneath the Search menu in the toolbar, as shown below.

  2. Find the section named Mutated Gene in the lower half of the form. Type the symbol of a mutated gene into the text box. For example, type Shh.

  3. Find the Sorting and output format section at the bottom of the form. Change the Return data type button to Assay Results.

  4. Click the Search button.

  5. The results will list assays using Shh mutants. The specific allele used to obtain each result will be listed in the Mutant Allele(s) column. Click on any link in the Result Details column to view details for a specific assay. If you don't see the Mutant Allele(s) column, return to the Gene Expression Data Query Form and select the Assay Results button as described in step 3.

  6. On the results page:

    • column 1 lists the gene in which expression was assayed;
    • column 2 describes the type of assay;
    • column 3 links to details for each assay;
    • column 4 lists the combination of alleles in the specimen used for each assay;
    • column 5 describes the age of the specimen;
    • column 6 describes the anatomical structure examined in the assay;
    • column 7 tells whether or not expression was detected.
