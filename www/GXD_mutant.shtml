How do I find expression data in mouse mutants?

Overview
You can use the Gene Expression Data Search Form to find detailed expression data of greatest interest to you using a variety of query parameters.
Accessing the Gene Expression Data Search Form
On most MGI pages, you can access the Gene Expression Data Search Form from the Search menu as shown in the image at right.

For this tutorial, open the gxd' onClick="javascript:openNewWindow('gxd'); return false;"> Gene Expression Data Search Form in a new window. Scroll down this page for further instructions.


Example 1. Finding expression assays for Shh mutants
  1. Use the Standard Search query form.

  2. Navigate to the Mutant/wild type row, select the Specimens mutated in gene: option and type shh in the text box. This will return a list of expression assays done in mice which carry mutant Shh alleles.

    To restrict the output to the Shh gene only, enter shh in the Genes text box at the top of the page as well.

  3. Click Search.
  1. The results summary lists expression assays that have been done on mice carrying Shh mutant alleles. The specific allele used in obtaining each result will be listed in the Mutant Allele(s) column. To refine or change search parameters, click the Click to modify search button at the top of the page.

  2. Use the tabs along the top of the table to navigate to a list of unique genes with expression results, assays, or to view images (box).

  3. Sort the table by clicking on column headers, apply filters and/or download as a text or Excel file using the buttons just above the table (arrow).

  4. Click on any data link in the Result Details column to more pattern and level of gene expression, images, and details of the specific assay, including mouse specimen details and antibodies or probes used.
