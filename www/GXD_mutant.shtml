How do I find expression data in mouse mutants?
Overview
You can use the Gene Expression Data Query Form to search for detailed expression data of greatest interest to you using a variety of query parameters.
Accessing the Gene Expression Data Query Form
On most MGI pages, you can access the Gene Expression Data Query Form from the Search menu as shown in the image at right.

For this tutorial, open the WIFetch?page=expressionQF' onClick="javascript:openNewWindow('WIFetch?page=expressionQF'); return false;"> Gene Expression Data Query Form in a new window.


Example 1. Finding expression assays for Shh mutants
  1. Type shh into the Mutated Gene text box.

  2. Change the Return data type button to Assay Results.

  3. Click Search.

  1. Click on any link in the Result Details column to view details for a specific assay.
The results summary will list assays using Shh mutants. The specific allele used to obtain each result will be listed in the Mutant Allele(s) column. If you don't see the Mutant Allele(s) column, return to the Gene Expression Data Query Form and select the Assay Results button as described in step 2.
