How do I find references that report expression data for a gene at a specific age?

  1. Click searches/gxdindex_form.shtml"TARGET = "BLANK">here to open the Gene Expression Literature Query Form in a new window. The Gene Expression Literature Query Form is located beneath the Search menu in the toolbar, as shown below.

  2. Type a gene symbol into the Gene Symbol/Name text box. As an example, use Fgf8.

  3. Select an age from the Age selection list. For example, select 10.5 days post-conception.

  4. Click the Search button.

  5. The search results list references that have examined Fgf8 expression at embryonic day 10.5 (E10.5) as well as the assay types used in these studies.

  6. The table at the top of the page groups the references by developmental stage analyzed and assay type used. Click a numerical link to retrieve a list of these references.

  7. Scroll down the page. The second table lists references that analyze Fgf8 expression. The numerical links indicate the number of results for each reference. Numerical links that are followed by an asterisk indicate that detailed assay results from that reference have been entered into the database. Click a numerical link to view a table that summarizes which ages were analyzed and which assay types were used to examine expression of that gene in that reference.
