How do I find references that report expression data for a gene at a specific age?

  1. Click searches/gxdindex_form.shtml"TARGET = "BLANK">here to go to the Gene Expression Literature Index Query Form. The Gene Expression Literature Index Query Form is located beneath the Search menu in the toolbar, as shown below.

  2. Type a gene symbol into the Gene Symbol/Name text box. As an example, use Fgf8.

  3. Select an age from the Age selection list. For example, select 10.5 days post-conception.

  4. Click the Search button.

  5. The search results list references that have examined Fgf8 expression at embryonic day 10.5 (E10.5) as well as the assay types used in these studies.

  6. The table at the top of the page records the number of references at each developmental stage by assay type. Click a numerical link to retrieve references.

  7. Scroll down the page. The second table lists references for Fgf8. The numerical links indicate the number of index results for each reference. Click a numerical link to view a table of index results by age and assay type. Numerical links followed by an asterisk indicate that detailed assay results exist for that reference.
