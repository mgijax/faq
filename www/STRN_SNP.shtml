How do I find all known coding SNPs in a specific genomic location for one or more strains?

  1. Click here to open the Mouse SNP Query Form in a new window. The Mouse SNP Query Form is located beneath the Search menu, as shown below.

  2. From the Available Strains list, select the following strains. To select all at once, hold down the Ctrl key on Windows machines while selecting each strain. On Macintosh machines, hold down the Command key.

    • 129X1/SvJ
    • C57BL/6J
    • DBA/2J

  3. Click the Add button to move strains to the Selected Strains list.

  4. In the SNP attributes section of the page, find the dbSNP Function Class pick list. Select both Coding-NonSynonymous and Coding-Synonymous at once by holding down the Ctrl key on Windows machines. On Macintosh machines, hold down the Command key.

  5. Find the Map position section of the page. In the Marker range text boxes at right, type Hnf4g in the first text box, and Car3 in the second.

  6. Click the Search button.

  7. A new browser window will open displaying coding SNPs identified between the Hnf4g and Car3 genes on chromosome 3.

  8. To view flanking sequence for a SNP, click any link to MGI SNP Detail in the SNP ID column at left.

  9. To view SNPs associated with all strains assayed, return to the Mouse SNPs Query Results page and click the Hnf4g link in the Gene: dbSNP Function Class column.

  10. On the Hnf4g Gene Detail page, find the Polymorphisms section and click the number in parentheses next to SNPs within 2kb(x from dbSNP build y).
