How do I search for a gene using an Entrez, Ensembl, VEGA, or other accession ID?

Overview
You can use the Search Box to find information about genes, markers or accession identifiers in MGI. The Search Box returns a list of genes, markers, or accession IDs that contain your entry.
Accessing the Search Box
On most MGI pages, you can find the Search Box in the upper right corner, as shown in the image.

For this tutorial, open the homepages/index.shtml"target = "blank">MGI home page in a new window or tab.


Example 1. Finding the gene associated with Entrez Gene accession identifier 18505
  1. Type 18505 into the search box.

  2. Click Search.
  1. The Search Results page lists symbols, names, and accession identifiers containing 18505. Click the link to MGI Marker Detail to retrieve the Gene Detail page for the Pax3 gene.
