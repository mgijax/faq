How do I search for a gene using an Entrez, Ensembl, VEGA, or other accession ID?

  1. Click homepages/index.shtml"TARGET = "BLANK">here to open the MGI home page in a new window.

  2. Type an accession ID into the search box in the upper right hand corner of the web page. As an example, use 18505.

  3. Click the Search button.

  4. On the Search Results page, find an entry described as "Gene" in the Type column.

  5. Click a link to MGI Marker Detail in the MGI Links column to retrieve a Gene Detail page. In this example you will retrieve the Gene Detail page for the Pax3 gene.

  6. For a list of accession IDs, click here to go to the Batch Query tool. The Batch Query tool is located beneath the Search menu, as shown below.

  7. Using the MGI Batch Query, you can enter a list or upload a file of official MGI gene/marker symbols, or gene, transcript, or sequence IDs from MGI, Ensembl, Entrez, Uniprot or other sources. You can choose to return gene symbols and names, genomic location, Gene Ontology or Mammalian Phenotype terms, MGI allele IDs, or IDs from other sources.

  8. As a quick example, type these two accession identifiers into the text box:

    18505
    18508

  9. For Input Identifier Type, select Entrez Gene ID.

  10. Click the Search button.
