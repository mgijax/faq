How do I display my mouse gene expression data in the context of mouse pathways?
Overview
You can use the Biochemical Pathways page to access mouse genetic and genomic data in the context of physiological and cellular processes.
Accessing the Biochemical Pathways page
On most MGI pages, you can access the Biochemical Pathways page by clicking the appropriate tab as shown in the image at right.

For this tutorial, open the homepages//minihome_BioPath.shtml' onClick="javascript:openNewWindow('homepages/minihome_BioPath.shtml'); return false;"> Biochemical Pathways page in a new window.


Example 1. Finding pathway data associated with expression data
  1. Click the link to the Overlay experimental data onto the Pathway Tools Omics Viewer.
  1. Select Mus musculus C57BL/6J from the Select a dataset drop-down menu.

  2. Click the Choose file button to upload a file. This file should be tab-delimited text with gene symbols in the first column and expression level in the second and all subsequent columns.

  3. Type the number 1 into the Data column (numerator in ratios) box.

  4. Click the Submit button located at the bottom of the page.
