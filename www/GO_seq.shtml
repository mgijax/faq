How do I obtain FASTA sequence files for genes with GO annotations in common?
Overview
You can use the Mouse Sequence Query Form to retrieve sequences from a specific region of the mouse genome, or sequences associated with a human disease, mouse phenotype, or function.
Accessing the Mouse Sequence Query Form
On most MGI pages, you can access the Mouse Sequence Query Form from the Search menu as shown in the image at right.

For this tutorial, open the searches/sequence_form.shtml' onClick="javascript:openNewWindow('searches/sequence_form.shtml'); return false;"> Mouse Sequence Query Form in a new window.


Example 1. Finding all polypeptide sequences annotated to Gene Ontology term "histone deacetylation"
  1. Select Polypeptide from the Sequence Type pick list.

  2. Type the term histone deacetylation in the Gene Classifications text box.

  3. Click Search.
  1. Click the Select all button to select all sequences.

  2. Click the Go button to download the selected sequences in FASTA format.
