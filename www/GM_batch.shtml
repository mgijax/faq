How do I retrieve Entrez, Ensembl, VEGA, or other IDs for a list of genes?

  1. Click here to go to the Batch Query tool. The Batch Query tool is located beneath the Search menu, as shown below.

  2. Using the MGI Batch Query, you can enter a list or upload a file of official MGI gene/marker symbols, or gene, transcript, or sequence IDs from MGI, Ensembl, Entrez, Uniprot or other sources. You can choose to return gene symbols and names, genomic location, Gene Ontology or Mammalian Phenotype terms, MGI allele IDs, or IDs from other sources.

  3. As a quick example, type these two gene symbols into the text box:

    Pax6
    Pax3

  4. For Input Identifier Type, select Current Gene/Marker Symbol.

  5. In the Output section, check the boxes next to Ensembl ID, Entrez Gene ID, or VEGA ID.

  6. Click the Search button.
